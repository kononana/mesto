# Проект: Место

Интерактивная страница, куда можно добавлять фотографии, удалять их и ставить лайки.

### Обзор технологий, использованных в проекте

- javascript
- flexbox
- grid
- организация файлов по БЭМ
- адаптиная верстка по макету
- медиазапросы
